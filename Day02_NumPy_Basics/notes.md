# Day 2: NumPy Basics

## What is NumPy?
NumPy is a Python library used for numerical computing.
It is faster than Python lists and is used in Machine Learning.

## Why NumPy is important for ML
- Handles large numerical data
- Supports multi-dimensional arrays
- Used by Pandas, Scikit-learn, TensorFlow

## Topics Covered
- NumPy array
- 1D and 2D arrays
- Indexing and slicing
- Basic operations
